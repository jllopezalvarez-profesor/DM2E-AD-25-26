Este ejemplo procesa el document xml-files/plants-catalog.xml para:
- Buscar todas las zonas que hay en el documento
- Para cada zona:
    - Contar cuantos productos hay con un precio mayor que 5$
    - Contar cuantos productos hay con un precio menor que 5$
    - Contar cuantos productos hay con valor "Shade" en el elemento "light"
    - Contar cuantos productos hay con valor "Sunny" en el elemento "light"
    - El resto de valores de "light" (Mostly shady, Sun or shade, ...) se ignoran, no se cuentan
